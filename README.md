🏢 Dormitory Management System (ระบบจัดการหอพักแบบครบวงจร)
ระบบบริหารจัดการหอพักนักศึกษาและอพาร์ตเมนต์ พัฒนาขึ้นเพื่อแก้ไขปัญหาการจัดการเอกสารแบบ Manual โดยเปลี่ยนมาเป็นระบบ Digital ที่ช่วยลดข้อผิดพลาดในการเช็คสถานะห้องว่าง การคำนวณบิล และการเก็บประวัติผู้เช่า
✨ Features (ฟีเจอร์หลักของระบบ)
ระบบถูกออกแบบมาให้ครอบคลุมการทำงานจริงของธุรกิจหอพัก ประกอบด้วย:
📊 Room Availability Dashboard: หน้าจอแสดงสถานะห้องพักแบบ Real-time (ว่าง/ไม่ว่าง) แยกตามสีชัดเจน
👥 Tenant & Employee Management: ระบบจัดการข้อมูลผู้เช่าและพนักงานในหอพัก
📅 Booking & Contracts: ระบบจัดการการจองห้องพัก และบันทึกสัญญาเช่า (ค่ามัดจำ, ค่าประกัน)
💸 Billing & Payments: ระบบคำนวณค่าน้ำ-ค่าไฟ และค่าเช่าอัตโนมัติ พร้อมระบบติดตามสถานะการชำระเงิน
🔧 Maintenance Requests: ระบบแจ้งซ่อมแซมและจัดการสถานะการซ่อมบำรุง
🖨️ Print & Search: ระบบค้นหาข้อมูลห้องพักและพิมพ์ใบเสร็จ (อยู่ระหว่างการพัฒนาเพิ่มเติม)
🛠️ Database Design & Tech Stack
Frontend: HTML, CSS (Bootstrap)
Backend: PHP
Database: MySQL (Relational Database Design with Normalization)
